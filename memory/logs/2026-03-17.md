### Root Cause of Golden Rate = 0
`validation.py` pre-fetch skip checked `cpc_usd is not None AND search_volume is not None`. Deferred keywords from keyword_extractor.py have `cpc_usd=0.0, search_volume=0` — both not None. So 975/977 keywords were SKIPPED from cache/API lookup. `_all_metrics` was empty. All went UNSCORED. Fixed by adding `metrics_source not in ("deferred","none_configured","")` to the condition.

### Root Cause of vector_store L2 Bug
LanceDB brute-force `.search()` ALWAYS uses L2 distance regardless of `metric="cosine"` in `create_table()`. The `metric` parameter only affects explicitly created IVF indexes. Old threshold `distance < (1-0.85)=0.15` would only catch vectors with cosine similarity > 0.9985 (nearly identical). Fixed to `distance < sqrt(2*0.15) ≈ 0.548`, which correctly captures cosine similarity ≥ 0.85 on normalized vectors.
